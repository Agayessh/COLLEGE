Let's Play BlackJack!



Gameplay:

Once the website load, a player can click on the button for a new game
Once the player clicks on the new game button, the player will be given 2 random cards
The player has the option to hit or stay
Hit means to add another card
Stay means the player is happy with the value of the cards at hand
The goal of the game is for the player to have a total card value of 21 or what we call a BlackJack
If the player exceeds 21, it is a Bust and the game will end
If the player exceeds 21, clicked the stay button, or get a black jack, all buttons will be disabled except for the new game


Requirements:

Use arrays to contain the cards
Use the Math.random() method to generate and add a random card for the player when it clicks the hit button
each card should only be shown once for each round, There should be no duplicate card that will be displayed
